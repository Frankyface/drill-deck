# drill-deck

A rugby club's shared drill library, as a touch-first mobile app. Coaches build a
searchable database of drills — tagged by category, skill focus, player count, and
equipment — with 2D pitch diagrams (animated playback planned), progressions,
session plans, and post-session reviews.

Built with **Expo (React Native + TypeScript)** and **Supabase**.
